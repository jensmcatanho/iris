§ SaveLog()

void Logger::SaveLog ( )

Saves all the log messages in a file.

Remarks
Since the whole API is very efficiency-dependent, the log messages are only written to a file when the program finishs it's execution. File reading and writing has a very expensive and unnecessary cost to deal with during bottlenecks of exectuion.
